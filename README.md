This is a SaaS Ai application where users can log in and access features on the website.

Development Tools:
- React.js and Tailwind Css (Frontend)
- Postgre SQL (Neon Database)
- Express (backend)
- Node.js (backend)
- Cloudinary (image storage)
- Gemini API (response generation)
- Clip Drop (image generation)
- Clerk (dynamic user handling)
- Vercel (deployment)
